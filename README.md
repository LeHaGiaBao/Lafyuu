- Mock Data

## UI design available here: [Behance](https://www.behance.net/gallery/93477733/FREE-eCommerce-UI-Kit)

## Installing

Clone the repository using following command or download

```
git clone https://github.com/LeHaGiaBao/Lafyuu.git
```

To install dependency

```
yarn install
cd ios && pod install
```

To start an app with the target environment

```
# iOS
npm run ios-staging // For staging app
npm run ios-dev // For development app
npm run ios-prod // For production app

# Android
npm run android-staging-debug // For staging app
npm run android-dev-debug // For development app
npm run android-prod-debug // For production app
```
